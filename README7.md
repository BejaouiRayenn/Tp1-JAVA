Exercice 7 :
 Tester si un Nombre est Premier ou pas Ce programme Java vérifie si un entier donné en argument via la ligne de commande est un nombre premier. Un nombre est premier s'il possède exactement deux diviseurs distincts et positifs : 1 et lui-même.
VOICI la trace d 'execution :
C:\JAVA\TP>java EX7 12
12 n'est pas un nombre premier.

C:\JAVA\TP>java EX7 111
111 n'est pas un nombre premier.

C:\JAVA\TP>java EX7 3
3 est un nombre premier.
