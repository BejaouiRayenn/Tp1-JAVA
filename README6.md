Exercice 6 : Détermination du Maximum de N Entiers

Ce programme Java lit un nombre fixe d'entiers (six dans cet exercice) passés en argument via la ligne de commande. Il détermine ensuite la plus grande valeur parmi ces six entiers et l'affiche.
Voici la trace d'exécution pour un exemple :
C:\JAVA\TP>java EX6 4 80 5 1 0 40
Le maximum est = 80
