# Exercice 5 : Calcul de la Factorielle

Ce programme Java calcule et affiche la factorielle d'un entier passé en argument depuis la ligne de commande.

## Code Source
Le code est disponible dans le fichier `Ex5.java`.

## Trace d'Exécution (CMD)

Pour un entier donné, la trace d'exécution est la suivante :

```bash
C:\JAVA\TP>javac Ex5.java
C:\JAVA\TP>java Ex5 4
La factorielle de 4 est 24
